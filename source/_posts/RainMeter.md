#### 配置

###### 系统要求

Rainmeter将在Windows 7及更高版本上运行。同一安装程序可以在32位或64位系统上使用，并且所有外观和功能都与两种体系结构兼容。

Rainmeter不适用于非Windows系统（包括Mac OS和Linux），并且没有计划支持这些操作系统。



###### 下载

首先从[Rainmeter.net](http://www.rainmeter.net/)下载Rainmeter安装程序。

Rainmeter有两个版本：当前版本的**最终**版本和下一个版本的**beta**版本。

Beta版通常每周发布一次，而最终版本则每隔几个月发布一次。尽管带有“ beta”标签，Rainmeter的每个发行版都是稳定的，并且与以前的版本向后兼容。

唯一的区别是，在下一版本正式发布之前，（很少）可以更改或删除Beta功能。

对于大多数希望获得最新功能和错误修复的用户，这里建议使用Beta版。

但是，如果不想冒险使用在将来的版本中可能由于更新而无法正常使用的Beta功能，建议使用最终版本。



下载过程据了解可能会非常慢，所以这里建议通过迅雷进行下载安装

{% asset_img 1.jpg Downloading image %}

安装将执行以下操作：

- 将程序安装到默认**文件夹**或选定的**Destination文件夹**。
- 为.rmskin文件扩展名和[Rainmeter Skin安装程序](https://docs.rainmeter.net/manual/installing-skins/)创建Windows文件关联。
- 为Rainmeter创建Windows“开始”菜单项。
- 为“启动/雨表”创建“ Windows开始”菜单项，因此Rainmeter从Windows开始。可以通过**在启动时**取消选中**Launch Rainmeter**来禁用此功能。
- 在安装结束时启动Rainmeter。

#### 额外操作：

如果您跟我一样，c盘的空间已经爆红的话，建议您进行接下来的操作：

将皮肤文件夹移到d盘

当您安装成功后，会发现任务栏中多了个水滴的图像：

{% asset_img 6.jpg Rainmeter image %}

接下来右击编辑设置，将文件路径修改为您指定的路径

{% asset_img 7.jpg Edit image_1 %}

{% asset_img 12.png Edit image_2 %}

首次运行Rainmeter时，它将在[默认文件位置](https://docs.rainmeter.net/manual/installing-rainmeter/#DefaultFileLocations)创建外观和设置文件夹。

默认文件夹分布如下：

**皮肤文件夹：**
`C:\Users\YourName\Documents\Rainmeter\Skins` 

**设置和布局文件夹：**
`C:\Users\YourName\Appdata\Roaming\Rainmeter` 

然后右击刷新全部，再点击关于-> 版本就会看到皮肤路径已经修改完了

{% asset_img 14.png Completed image %}

{% asset_img 8.jpg Exit image %}

注：修改过后要把rainmeter 退出，然后重新打开，

否则可能会没有修改的结果显示
